Instruções
==========

Para instalar:
```sh
git clone https://github.com/ITAbits/site.git
cd site
npm install
```

Para dar deploy da master

```sh
npm run deploy
```


Dependencias
============

[React](https://reactjs.org/)

[Semantic UI React](https://react.semantic-ui.com/introduction)


FAQ
===
